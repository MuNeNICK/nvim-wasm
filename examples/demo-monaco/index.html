Neovim + Monaco (headless WASM)
waiting mode: - cursor: - buf: - Serve with COOP/COEP so SharedArrayBuffer works.

Drop `nvim.wasm` and `nvim-runtime.tar.gz` into this folder (copied from `examples/demo` or built outputs). Keys go to Neovim; Monaco renders buffer + cursor via buffer events, similar to vscode-neovim.
